# 幼稚园选校参考 (Kindergarten)

香港幼稚园不属于政府的免费学位派位体系,选校以家长**直接向学校报名**为主,因此
"竞争机制"较小学/中学松散,但热门学校(尤其国际及一条龙)仍非常抢手。

## 一、学校类型

| 类型 | 说明 | 学费 |
|---|---|---|
| 参加幼稚园教育计划的非牟利幼稚园 | 政府向合资格非牟利幼稚园直接资助,半日制名额一般可达"免费"或仅收少量杂费 | 半日制多为免费或低收费;全日制/长全日制需补差额 |
| 私立独立幼稚园 | 不参加资助计划,自行定价,课程与师生比弹性较大 | 数万至十余万港币/年 |
| 国际幼稚园 | 多采用英语或双语、国际课程理念(如蒙特梭利、Reggio、IB-PYP 取向) | 通常十余万至二十余万港币/年,部分另收债券/资本费 |

补充概念:

- **幼稚园教育计划(Kindergarten Education Scheme)**:政府的资助机制,核心是为合资格
  非牟利幼稚园的**半日制**课程提供资助。全日制、长全日制家长通常需补付差额。
- **一条龙 / 直属 / 联系**:部分幼稚园与某小学有衔接关系,读完幼稚园可较有把握升入
  关联小学。对希望减少小一选校压力的家庭,这类幼稚园价值高——但要核实衔接是
  "保证"还是"优先考虑",两者差别很大。
- **半日制 vs 全日制 / 长全日制**:影响家庭日常照顾安排与费用,需结合双职父母情况。

## 二、报名机制与时间线

幼稚园没有统一派位。一般机制:

- **K1(约 3 岁入读)是主要入学点。** 报名通常在入学前一年的下半年(约 9–11 月)进行,
  部分热门学校更早。
- 多数学校流程为:递交报名表 → 面见(家长 + 幼儿,形式多为轻松互动/游戏观察)→ 收生通知。
- 国际及热门私立学校可能有**轮候名单(waitlist)**,需尽早递表。
- K2、K3 插班视乎学校有无空缺,机会较少。

**注意:具体报名月份每年、每校不同,务必以学校官网及教育局《幼稚园概览》为准,
作答前用 web search 核对当年信息。**

## 三、选校要点

针对幼稚园阶段,评估时特别关注:

1. **教学理念**:游戏化/愉快学习 vs 较早引入读写算。3–6 岁阶段,理念与孩子性格的
   契合往往比"名气"更重要。
2. **语言环境**:粤语、英语、普通话的比重。若家庭目标是英语为主的小学/国际路线,
   幼稚园的语言环境会影响衔接顺畅度。
3. **衔接关系**:是否一条龙/直属某小学;若家长极看重减少小一压力,这是关键加分项。
4. **地理位置**:幼儿年纪小,通勤时间应从严控制。
5. **作息制度**:半日/全日是否配合家庭照顾安排。
6. **设施与师生比**:活动空间、户外场地、每班人数与教师配置。

## 四、给家长的提醒

- 幼稚园选择对长远升学的"锁定效应"通常弱于小学/中学(除非是强衔接的一条龙)。
  不必把幼稚园决定承担过重的"决定一生"压力。
- 若计划走本地资助小学路线,幼稚园阶段可同步研究居住地所属**小学校网**(见
  primary 参考文件),提早布局。
